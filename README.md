# Game
Objective -3

Dodge the Falling Blocks
What it is:
A small console game where you control a player (^) at the bottom and try to dodge falling blocks (#). Survive as long as you can and get a high score.
How to play:
-Press a to move left
-Press d to move right
-Avoid hitting the falling blocks
Scoring:
-Every block you dodge gives 1 point
-Game ends if a block hits you

Can be upgraded later with more blocks or levels
